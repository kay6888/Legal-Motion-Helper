# Legal Motion Helper

A fast, polished web app for drafting legal motion text with stronger validation and clearer generation flow.

## What improved

- Responsive single-page UX for motion drafting.
- Structured form with required-field checks and actionable validation errors.
- Clear loading, success, and error states during generation.
- Improved generated output structure (caption, introduction, facts, legal argument, relief, conclusion, signature block).
- One-click copy and `.txt` download actions.
- Draft autosave in local storage for faster recovery and iteration.

## Usage

1. Open `/home/runner/work/Legal-Motion-Helper/Legal-Motion-Helper/index.html` in a browser.
2. Complete the form fields.
3. Click **Generate Motion** (or press `Ctrl+Enter` / `Cmd+Enter`).
4. Review and customize the generated draft for jurisdiction-specific requirements.

> This tool assists drafting and does not provide legal advice.

## Files

- `index.html` – UI layout and interaction points.
- `styles.css` – responsive styles and visual polish.
- `app.js` – validation, generation logic, autosave, and output actions.
